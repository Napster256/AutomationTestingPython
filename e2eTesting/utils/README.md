# Утилиты

Этот каталог содержит вспомогательные функции и утилиты повторного использования.

## Назначение

* Использовать общую логику для всех тестов
* Сохранять файлы тестов чистыми и целенаправленными

## Примеры

* Генераторы данных
* Вспомогательные утилиты
* Функции форматирования

## Рекомендации

* Должны быть независимыми от пользовательского интерфейса
* Не должно содержать специфичных для Selenium команд (за исключением случаев необходимости)
* По возможности сохраняйте функциональность в чистом виде
