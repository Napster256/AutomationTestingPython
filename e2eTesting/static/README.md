# static

Директория для хранения статических ресурсов.

Может использоваться для:
- Тестовых файлов (PDF, изображения)
- Данных для загрузки
- Файлов, используемых в upload/download тестах
- Вспомогательных mock-ресурсов

Не является частью стандартной структуры Selenium,
добавляется при необходимости проекта.
