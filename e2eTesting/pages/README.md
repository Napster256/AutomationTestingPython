# Объекты страницы

Этот каталог содержит реализации объектной модели страницы (POM).

## Назначение

* Инкапсулировать логику пользовательского интерфейса и селекторы
* Улучшить читаемость тестов и удобство сопровождения
* Уменьшить дублирование кода

## Структура

* Сгруппированы по функциям (аутентификация, именованные главные страницы)
* Каждая страница представляет собой экран пользовательского интерфейса или компонент

## Рекомендации

* Не включайте утверждения в объекты страницы
* Используйте методы повторно и просто
* Храните селекторы внутри файла подкачки
